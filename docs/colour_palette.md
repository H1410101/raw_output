# Raw Output Functional Tokens

The visual identity is defined by functional tokens organized into environmental foundations, hierarchical typography bands, and status indicators.

## Environmental Foundations
- **background-1**: `#020203` (The Abyss / Primary Background)
- **background-2**: `#130F1A` (Deep Space / Panels & Glass Tint)

## Lower Typography Band (Darker)
This band provides hierarchical depth for UI text, moving from deep decorative support to standard muted visibility.
- **lower-band-1**: `#3D527A` (Deep Muted / Decorative Text)
- **lower-band-2**: `#496C93` (Supporting / UI Metadata Text)
- **lower-band-3**: `#5485AB` (Standard Muted Text)

## Upper Typography Band (Lighter)
This band facilitates high-visibility interface text and secondary accents.
- **upper-band-1**: `#93CACD` (Soft Cyan / Normal Text)
- **upper-band-2**: `#AAD8D8` (Pale Mint / Secondary Accent)
- **upper-band-3**: `#C1E6E3` (Frost Mint / Primary Accent)

## Highlights & Callouts
- **highlight-font-1**: `#E0D9BB` (Champagne / Peak Highlight Text)
- **highlight-font-2**: `#FFFFFF` (Absolute White / Critical Focus)
- **highlight-bg-1**: `#252B46` (Midnight Blue / Interactive Elements)

## Status & Tagging
- **status-medium**: `#F7B989` (Sandy Orange / Neutral Warning)
- **status-hard**: `#BC5B4C` (Deep Coral / Error States)
